# Student Result Management System

A simple Java console project using Oracle JDBC.

## How to Run

1. Make sure you have Oracle DB and `ojdbc17.jar`.
2. Compile:
   javac -cp ojdbc17.jar *.java
3. Run:
   java -cp .;ojdbc17.jar Main

## Features

- Admin Login
- Add/View/Update/Delete Students
- Add/View/Update/Delete Results
- Display Total/Average/Grade
